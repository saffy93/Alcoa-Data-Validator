# ALCOA+ Data Validator



**A simple Python tool to validate data integrity using ALCOA+ principles**



This project demonstrates practical application of ALCOA+ (Attributable, Legible, Contemporaneous, Original, Accurate, Complete, Consistent, Enduring, Available) — core principles used daily in regulated histology laboratories and clinical environments.



## Features

- Validates data records against all 9 ALCOA+ principles

- Returns clear ✅ PASS / ❌ FAIL messages with specific failure reasons

- Easy to extend for real lab or clinical trial data
